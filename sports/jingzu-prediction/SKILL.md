# 竞足赛前预测系统

## 触发条件
用户提供竞足比赛对阵和59itou数据后，执行预测分析。

## ⛔ 核心约束
- **严禁构建脚本** — 纯经验推演，不使用Python/JavaScript进行模型计算
- **严禁总结数据** — 59itou详情数据必须原样展示
- **严禁主观臆断** — 所有结论必须有数据锚点
- 禁用词：必胜、稳胆、铁胆、绝对、肯定
- 禁止编造未提供的伤病/战术细节
- 优先级：指数信号 > 盈亏结构 > 基本面 > 联赛常识
- 矛盾处理：当基本面与指数冲突时，以指数/盈亏信号为准

## 数据获取流程

### 竞足列表
```
https://kt.59itou.com/192/jingcai/
```
从北单列表 `883/danchang/` 获取matchID（共用），因竞足列表页无matchID。

### 竞足详情
```
https://kt.59itou.com/379/match3/?matchid={matchid}&lotteryId=90&lottery_style=jczq
```
逐Tab获取数据（Tab索引：3战绩/2阵容/4欧指/5亚指/8排名/9盈亏/7推荐/6情报/0直播），每次切换等待1秒，用 `document.body.innerText` 获取原始内容。

## 五步推演框架

### 1️⃣ 联赛基因定位
- 判断类型：大球（德甲/荷甲/日职联等场均>2.7）｜小球（意甲/法甲等场均<2.4）｜中性（英超/西甲等）
- 提取特征：该联赛典型比分分布、主场优势强度、平局高频场景
- 将联赛特性作为权重调节器，非决定因素

### 2️⃣ 指数信号解码（核心步骤）
- **欧指**：平赔↓+升平机构≥20家 → 机构真实防范平局
- **亚盘**：半球盘上盘高水（≥1.90）+赢盘率<50% → 主胜阻力大
- **赔付**：庄家"平/负"盈利+"胜"亏损 → 机构规避主胜风险
- 标注矛盾点：如"升主胜26家但赔付主胜仅10家"

### 3️⃣ 基本面交叉验证
- 状态：连胜/连败是否与指数方向一致？
- 排名：积分差距是否被指数弱化？
- 进球分布：0-1球占比是否支持小比分？

### 4️⃣ 市场热度校准
- 必发：交易量占比 vs 庄家盈亏，判断热度陷阱
- 冷热指数：|交易量占比-概率|>15% → 价值方向在冷门方

### 5️⃣ 多维合成推演
- 权重：指数信号(40%) + 盈亏结构(30%) + 基本面(20%) + 联赛特性(10%)
- 生成比分概率排序（仅限数据高频出现比分）
