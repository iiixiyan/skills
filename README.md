## 📦 Skills

### manual-to-playwright-automation

将手工测试用例转换为 Playwright UI 自动化测试脚本。

**支持格式:** Excel / Markdown / JSON / XMind / TestLink XML / CSV

**核心能力:**
- 多格式用例解析
- 自动化可行性分析
- Page Object Model 自动生成 (TypeScript + Python)
- 优先级标签映射 (@P0/@P1/@P2/@P3)
- 转换总结报告
- 数据驱动测试、API 集成
